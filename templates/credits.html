Credits

This project was inspired by the Web of Science and the Court Listener Supreme Court Citation Networks Project.

The raw citation data was taken from Generative Dynamics of Supreme Court Citations: Analysis with a New Statistical Network Model by Schmid, Chen, and Desmarais (2021), and was initially created by Lissner and Carver using Court Listener data.

Unfortunately, the data is extremely patchy and only goes until 2003. More recent data is available on Court Listener, but it has so much data that it is impractical to search quickly (observe the slow speed of the Citation Network Project results), it contains many data artifacts that must be filtered (and further slow search), and has extremely patchy citation records for very old cases compared to Lissner and Carver.

Regardless, in the long term, it is likely a more powerful data source, and I hope to integrate it in the future.

This website extensively uses Cytoscape.js, a graph theory library for visualisation and analysis to generate its graphs ( Bioinformatics (2016) 32 (2): 309-311 first published online September 28, 2015 doi:10.1093/bioinformatics/btv557).
