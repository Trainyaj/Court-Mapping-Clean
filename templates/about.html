About the Supreme Court Citation Lookup

Welcome to the Supreme Court Citation Lookup! Using this tool, you can look up Supreme Court cases by name and build citation network graphs.

Be warned: the search tool is relatively sensitive to variations in case, spelling, punctuation, and abbrivation, and will often fail to find a case if the case name is entered incorrectly.

Also be warned: the citation database I am using is incomplete, so many citations will be missing (see Credits for more details). Some nodes (cases) will be blank. This is because they don't have names in the database I'm using. Sorry! The citation database also often has several different cases named the same (for example, there are two "Brown v. Board of Education"'s'), usually representing different administative steps in the case (for example, Brown 1954 vs. Brown 1955).

After you build a citation network around a given case (denoted in green), click on one of the surrounding cases, and the citation network will grow around that case!
